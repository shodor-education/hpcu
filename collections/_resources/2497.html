--- title: "Parallelization: Area Under a Curve" start-date: null end-date: null resource-url: "http://www.shodor.org/petascale/materials/UPModules/AreaUnderCurve/" description: "This module teaches: 1) How to approximate the area under a curve using a Riemann sum, 2) how approximating the area under a curve is used in solutions to scientific problems, 3) how to implement parallel code for Area Under a Curve (including versions that use shared memory via OpenMP, distributed memory via the Message Passing Interface (MPI), and hybrid via a combination of MPI and OpenMP), 4) how to measure the performance and scaling of a parallel application in multicore and manycore environments, and 5) how Area Under a Curve falls into the MapReduce \"dwarf\" (a class of algorithms that have similar communication and computation patterns).
